# 經費規劃

**執行者：** sbir-writer

協助細化與驗證 SBIR 經費，確保比例合規、ROI 達標。

## 經費規則

| 類別 | 上限 |
|------|------|
| 人事費 | ≤ 60% |
| 委外費 | ≤ 30% |
| Phase 1 總額 | ≤ 150 萬 |
| Phase 2 總額 | ≤ 1,200 萬 |
| ROI（3 年累計） | ≥ 1:3 |
