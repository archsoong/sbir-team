# 資格篩選

**執行者：** sbir-pm

判斷客戶是否符合 SBIR 申請資格。6 項硬性條件全部通過才能進入下一步。

## 判斷規則

| 項目 | 通過條件 |
|------|----------|
| 實收資本額 | < 1 億台幣 |
| 員工人數 | < 200 人 |
| 公司組織 | 公司或有限合夥（非分公司、行號） |
| 陸資 | 無陸資投資 |
| 欠稅 | 近 3 年無欠稅紀錄 |
| 違規 | 近 5 年無政府計畫重大違約 |
